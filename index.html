CREATIONS

Some creations I have done

Throughout my approximately four (4) years of programming experience, I have developed various creations and systems. For basic HTML-CSS-JS games, I have programmed TicTacToe, Color-Guessing Games, and other games which do not require object-rendering.

Minercaft Scripting API related, I have developed wrappers, command handlers, event handlers, custom mini-game systems. and more.




TICTACTOE

This TicTacToe game which I have developed is both a Player VS Player and a Player VS Bot where the bot's data (place delay and accuracy) can be edited.

The bot has an algorithm that it follows where it detects possible winning orientations of the player and stops them (the player) from winning easily.

COMMAND HANDLER

A command handler which manages the creation and execution of custom commands with the Scripting-API of Minecraft Bedrock Edition.

This command handler may be quite outdated due to the fact that it has not been maintained for quite some time now. However, the function of the system is simply the management of custom commands and their execution in order to allow dynamic parameters for various commands.
